# ASCII Art Table Parser

## Requirements

- Python 3.8+
- `requests` library
 
## Create a Python virtual environment:

```bash
python3 -m venv venv
```
Activate the virtual environment:

On macOS/Linux:

```bash
source venv/bin/activate
```

On Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install requests
````

---


1. Run the script:

```bash
python ascii_art_viewer.py
```

2. Enter the URL of the HTML page containing the ASCII art table when prompted:

```text
Enter the URL for the ASCII shape test case: https://example.com/ascii_table.html
```

The ASCII art will be printed in your terminal.
